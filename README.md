# 🛍️ Customer Shopping Behavior Analysis

## 📊 Project Overview
Complete end-to-end Data Analytics project
analyzing Customer Shopping Trends from
Retail Data of 3,900 customers using
SQL, Python and Power BI.

## 🛠️ Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (SQLite)
- Power BI Desktop
- Jupyter Notebook

## 📁 Project Files
- Customer_Shopping_Behavior_Analysis.ipynb
  → Complete Python + SQL Analysis
- customer_shopping_behavior.csv
  → Dataset (3,900 records, 18 columns)
- power_bi_dasboard.pbix
  → Interactive Power BI Dashboard

## 📌 Key Business Insights
- Total Customers: 3,900
- Total Revenue: $233,081
- Average Order Value: $59.76
- Best Season: Fall
- Top Category: Clothing
- Most Purchased Item: Jewelry
- Popular Payment Method: Credit Card
- Male Customers: 68%
- Top Spending Age Group: 36-50

✅ 15+ SQL Queries for data extraction

✅ Customer segmentation by age & gender
✅ Revenue analysis by category & season
✅ Payment method analysis
